About

Symbol Finder is a tool to help users brainstorm word & image associations for abstract concepts.


How to use it?
After selecting an organization, the user would input a CONCEPT, and then sf presents three-word clusters associated with the concept for the user to choose from. The user can gather images using the suggested words derived from the concept.

Tips

Choosing a Good Concept

Use an abstract concept, such as "knowledge" instead of "book." A book is tangible, whereas knowledge is intangible.

Number of Terms

Based on previous user studies, we recommend finding at least 20 terms for each concept. This will ensure a good variety of images for your concept!

When to Use SF

Use SF for brainstorming associations, searching for a range of symbols, & designing images that combine multiple concepts.
